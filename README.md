## 🚀 Quick Start

### 1. Install Python dependencies
```bash
pip install flask flask-cors pymongo bcrypt PyJWT python-dotenv
`````n
### 2. Start MongoDB
MongoDB must be running on localhost:27017

### 3. Start the backend
```bash
python server.py
`````n
### 4. Open Admin Panel
Open `index.html` in your browser.
